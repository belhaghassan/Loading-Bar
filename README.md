## Loading Bar

Welcome to the Loading Bar page!

## How to open

As this webpage is not an active on the World Wide Web you're
going to have to do it the old fashion way.

1. Download the index.html and styles.css files.

2. Open the folder they were downloaded to and double click index.html.

-Voila! It will open on your local machine.

## Description

This is a very simple webpage that shows a loading bar
on screen that takes 3 seconds to fill. And it resets the
bar back to zero when the Reset button is clicked. Enjoy?
